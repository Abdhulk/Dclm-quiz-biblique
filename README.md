# Biblical Quiz Game

## Overview
The Biblical Quiz Game is an interactive quiz application designed to test players' knowledge of biblical facts, stories, and teachings. The game presents users with a series of questions from various sections of the Bible, providing a fun and educational experience for players of all ages.

## Features
- **Multiple Choice Questions**: Players can choose from multiple answers for each question.
- **Timed Quizzes**: Players can test their knowledge under time constraints.
- **Score Tracking**: Players receive scores based on their correct answers and can see their progress after completing the quiz.
- **User-Friendly Interface**: Designed with accessibility in mind, the game is easy to navigate for users of all skill levels.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB (for storing questions and user scores)

## Installation
To run this project locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/Abdhulk/dclm-quiz-biblique.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dclm-quiz-biblique
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
5. Open your browser and go to `http://localhost:3000`.

## How to Play
1. Start the quiz and read each question carefully.
2. Select the answer you believe is correct.
3. Check your score after completing the quiz.
